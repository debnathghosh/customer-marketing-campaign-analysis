# Customer Marketing Campaign Analysis

## Overview

This project analyzes customer marketing data to understand purchasing behavior and identify meaningful customer segments. Using exploratory data analysis and clustering techniques, the project provides insights that can help businesses design more effective marketing strategies.

The project combines **Python-based data analysis** with **machine learning clustering methods** and **Power BI dashboards** for visualization.

---

## Objectives

* Perform exploratory data analysis on customer marketing data
* Clean and preprocess the dataset
* Identify customer segments using clustering techniques
* Analyze purchasing behavior and campaign responses
* Visualize insights using an interactive dashboard

---

## Dataset

The dataset contains information about customer demographics, income, purchasing patterns, and marketing campaign responses.

Example attributes include:

* Customer demographics
* Income levels
* Product purchase history
* Marketing campaign responses
* Spending patterns

## How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/debnathghosh/customer-marketing-campaign-analysis.git
```

### 2. Navigate to the project folder

```
cd customer-marketing-campaign-analysis
```

### 3. Install required libraries

```
pip install pandas numpy matplotlib seaborn scikit-learn
```

### 4. Run the notebooks

Open Jupyter Notebook and run:

* `data_preprocessing.ipynb`
* `clustering_analysis.ipynb`
